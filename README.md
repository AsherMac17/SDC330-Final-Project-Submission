The program that I created for this project is a to-do-list manager.  It is a task management application where users can add, remove, and view tasks that they assign into the program. Users can keep track of their tasks throughout the day with this application. The program will include an interface/abstract class. The program will demonstrate polymorphism and composition and store data into a database. 

Tasks:
Task 1: Setup the folder in VScode to start working on the project.
Task 2: Plan out the classes and how they will function with the program.
Task 3: Implement the classes into the program and its functionalities. Write code that effeciently implements the project.
Task 4: Test and debug the code to make sure it works properly and everything is in check.
Task 5: Once it all works document the content and submit.

Project skills learned: 
Github control for storing code
Project planning and task management for the project
Debuging and writing advanced Java 

Language used: Java
